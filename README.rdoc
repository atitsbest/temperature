== Anforderungen

 * Aktuelle Temperatur für jeden Sensor anzeigen.
  - Ändert sich eine Temperatur, muss dazu die Seite nicht neu geladen werden.
 * Dauer seit der letzten Messung für jeden Sensor anzeigen.
 * Temperaturverlauf für jeden Sensor einzeln anzeigen
  - der letzten 24h
  - der letzten 7 Tage
  - des letzten Monats
  - des letzten Jahres
  - alle
 * Maximal/Minimal/Durchschnitts-Temperatur für jeden Sensor anzeigen.
